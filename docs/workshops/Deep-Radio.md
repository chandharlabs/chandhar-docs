# Deep Learning for Wireless

This workshop introduces the application of
**deep learning techniques to wireless communication systems**.

---

## Workshop Details

- **Duration:** 3–5 Days
- **Mode:** Hands‑On / Lab‑oriented
- **Audience:** Students, Researchers, PhD Scholars

---

## Topics Covered

- Introduction to AI/ML for wireless
- Dataset generation from SDRs
- ML‑based channel estimation
- AI‑assisted PHY and MAC
- Reinforcement learning for resource allocation

---

## Platforms Used

- SDR‑based testbeds
- Deep Radio framework
- Python‑based ML pipelines
